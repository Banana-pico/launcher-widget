# Launcher Widget

Windows 11 のデスクトップに常駐する、軽量でスタイリッシュなランチャーウィジェットです。

![Launcher Screenshot](https://raw.githubusercontent.com/hiza-NAS/launcher/main/screenshot.png) *(※プレースホルダー)*

## 🚀 特徴

- **超軽量動作**: C++ と Win32 API のみで構築。追加のランタイム（.NET, Electron 等）は不要です。
- **スタイリッシュなデザイン**: DWM（Desktop Window Manager）による角丸ウィンドウと半透過のモダンな UI。
- **カスタマイズ自在**: グリッドの行数、列数、ボタンサイズ、余白、色、透明度を自由に変更可能。
- **マルチページ対応**: 用途に合わせて複数のページを作成し、素早く切り替えられます。
- **多様なアクション**:
  - アプリの起動（.exe, ストアアプリ/UWP 対応）
  - ファイル・フォルダー・URL のオープン
  - Windows 設定（ms-settings）の呼び出し
  - キーボードマクロ（同時押し、順次押し）
  - メディア・音量コントロール
- **自動アイコン取得**: アプリのアイコンや Web サイトの favicon を自動的に取得して表示します。

## 🛠 操作方法

- **左クリック**: 設定されたアクションを実行します。
- **右クリック (ボタン)**: そのボタンの設定（名前、アクション、画像など）を編集します。
- **右クリック (背景)**: アプリケーション全体の設定やページの管理メニューを開きます。
- **SET ボタン**: 右上の `SET` からグリッド設定や外観設定を変更できます。
- **ページ切り替え**: `Page +` / `Page -` ボタン、またはページ名を右クリックして操作します。

## 📦 インストール

1. [Releases](http://192.168.1.102:3000/hiza-NAS/launcher/releases) から最新のインストーラー（`LauncherSetup.exe`）をダウンロードします。
2. インストーラーを実行して指示に従ってください。
3. 起動後、スタートアップへの登録オプションを有効にすると、PC 起動時に自動で常駐します。
